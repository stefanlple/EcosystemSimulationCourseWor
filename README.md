# Ecosystem-Simulation
## Displays predator-prey relationship between foxes and bunnies
### Features:
-Diagramms/Stats(Population size, Animal died...)  
-Hunting mechanism  
-Eating & Drinking  
-Reproducing  
-Music  
-Menu
